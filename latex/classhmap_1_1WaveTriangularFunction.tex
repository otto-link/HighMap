\doxysubsection{Detailed Description}
Wave triangular (x, y) function class. 

This class models a wave triangular function with specific noise wavenumbers, rotation angle, slant ratio, and phase shift. It extends the base \doxylink{classhmap_1_1Function}{Function} class.
