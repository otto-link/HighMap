\doxysubsection{Detailed Description}
A class to represent a multi-\/dimensional tensor. 

The {\ttfamily \doxylink{classhmap_1_1Tensor}{Tensor}} class is used to store and manipulate multi-\/dimensional arrays (tensors) of floating-\/point numbers. It supports element access, basic operations such as finding the minimum and maximum values, remapping the tensor values, and exporting the data to images.
