\doxysubsection{Detailed Description}
Provides functions for synthesizing new heightmaps from input arrays. 

\begin{DoxyAuthor}{Author}
Otto Link (\href{mailto:otto.link.bv@gmail.com}{\texttt{ otto.\+link.\+bv@gmail.\+com}}) 
\end{DoxyAuthor}
This header file defines functions for synthesizing new heightmaps by using various techniques such as non-\/parametric sampling, patch quilting, and blending. These functions allow for the generation of larger or modified heightmaps based on input data, using methods like patch stitching and random sampling.

Functions provided\+:
\begin{DoxyItemize}
\item {\ttfamily non\+\_\+parametric\+\_\+sampling}\+: Generates a new heightmap using non-\/parametric sampling.
\item {\ttfamily quilting}\+: Synthesizes a new heightmap by stitching patches from a set of input heightmaps.
\item {\ttfamily quilting\+\_\+blend}\+: Similar to {\ttfamily quilting}, but includes blending of patches.
\item {\ttfamily quilting\+\_\+expand}\+: Synthesizes a larger heightmap by expanding and stitching patches from an input heightmap.
\item {\ttfamily quilting\+\_\+shuffle}\+: Similar to {\ttfamily quilting\+\_\+expand}, but reshuffles patches to generate a new heightmap with the same shape.
\end{DoxyItemize}
