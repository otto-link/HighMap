Updates the point\textquotesingle{}s value based on bilinear interpolation from an array. 

This function updates the value of the {\ttfamily \doxylink{classhmap_1_1Point}{Point}} object by performing bilinear interpolation on the input {\ttfamily \doxylink{classhmap_1_1Array}{Array}} using the coordinates of the {\ttfamily \doxylink{classhmap_1_1Point}{Point}} and a given bounding box. The point\textquotesingle{}s coordinates are first normalized to the unit interval using the provided bounding box. Then, these normalized coordinates are scaled to the array\textquotesingle{}s dimensions and used to fetch the value from the array through bilinear interpolation.

If the normalized coordinates fall outside the bounds of the array, the point\textquotesingle{}s value is set to zero.


\begin{DoxyParams}{Parameters}
{\em array} & The input {\ttfamily \doxylink{classhmap_1_1Array}{Array}} from which the value is interpolated. The {\ttfamily \doxylink{classhmap_1_1Array}{Array}} should support bilinear interpolation. \\
\hline
{\em bbox} & Bounding box used for normalizing the {\ttfamily \doxylink{classhmap_1_1Point}{Point}}\textquotesingle{}s coordinates. This box is defined by a {\ttfamily \doxylink{structhmap_1_1Vec4}{Vec4}\texorpdfstring{$<$}{<}float\texorpdfstring{$>$}{>}} containing minimum and maximum values for both x and y dimensions in the format {\ttfamily \{xmin, xmax, ymin, ymax\}}.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
If the coordinates are outside the bounds of the array after scaling, the point\textquotesingle{}s value is set to zero. 
\end{DoxyNote}
