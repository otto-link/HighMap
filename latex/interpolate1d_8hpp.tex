\doxysubsection{Detailed Description}
Defines a 1D interpolation class using the GSL (GNU Scientific Library). 

\begin{DoxyAuthor}{Author}
\texorpdfstring{$\ast$}{*} Otto Link (\href{mailto:otto.link.bv@gmail.com}{\texttt{ otto.\+link.\+bv@gmail.\+com}}) 
\end{DoxyAuthor}
This file contains the {\ttfamily Interpolator1D} class and {\ttfamily Interpolation\+Method1D} enum, which provide a simple interface for performing 1D interpolation with various methods available in the GSL. The interpolation methods include Akima, cubic, linear, polynomial, and Steffen interpolation.
