\doxysubsection{Detailed Description}
A class for performing 1D interpolation using the GSL library. 

The {\ttfamily \doxylink{classhmap_1_1Interpolator1D}{Interpolator1D}} class provides an interface for performing 1D interpolation on a set of data points. The interpolation methods supported are defined by the {\ttfamily Interpolation\+Method1D} enumeration. The class uses the GSL library for the interpolation computations.
