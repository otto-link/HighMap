\doxysubsection{Detailed Description}
Class for performing curve interpolation on a set of points. 

The {\ttfamily \doxylink{classhmap_1_1InterpolatorCurve}{Interpolator\+Curve}} class allows interpolation over a set of points using a specified interpolation method (Bezier, B-\/spline, Catmull-\/\+Rom, or linear interpolation).
