\doxysubsection{Detailed Description}
Represents a line segment in 2D space. 

The {\ttfamily \doxylink{classhmap_1_1Edge}{Edge}} class defines a line segment using two {\ttfamily \doxylink{classhmap_1_1Point}{Point}} objects to represent the start and end of the segment. Additionally, it has a weight associated with the edge, and provides functionality to compute the length of the segment.
