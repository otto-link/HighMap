\doxysubsection{Detailed Description}
Header file defining basic vector and matrix manipulation classes. 

\begin{DoxyAuthor}{Author}
Otto Link (\href{mailto:otto.link.bv@gmail.com}{\texttt{ otto.\+link.\+bv@gmail.\+com}}) 
\end{DoxyAuthor}
This header file provides basic implementations for 2D, 3D, and 4D vector classes ({\ttfamily Vec2}, {\ttfamily Vec3}, {\ttfamily Vec4}) and a simple matrix class ({\ttfamily Mat}). These classes offer fundamental operations such as addition, subtraction, multiplication, division, dot products, and basic matrix manipulation capabilities.
