\doxysubsection{Detailed Description}
A class for generating fractal noise functions based on an underlying noise function. 

The {\ttfamily \doxylink{classhmap_1_1GenericFractalFunction}{Generic\+Fractal\+Function}} class generates fractal noise using an underlying base noise function. It allows customization of the fractal properties such as octaves, weight, persistence, and lacunarity.
