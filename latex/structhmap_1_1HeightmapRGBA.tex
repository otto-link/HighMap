{\footnotesize\ttfamily \mbox{\hyperlink{classhmap_1_1Heightmap}{Heightmap}} hmap\+::\+Heightmap\+RGBA\+::luminance (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Computes the luminance of an RGBA height map. 

This method creates a grayscale {\ttfamily Height\+Map} based on the luminance values calculated from the red, green, and blue channels of the RGBA height map. The luminance is computed using the standard formula\+: $
L =
0.299 \times R + 0.587 \times G + 0.114 \times B $.

\begin{DoxyReturn}{Returns}
A {\ttfamily Height\+Map} representing the grayscale luminance of the current RGBA height map.
\end{DoxyReturn}
