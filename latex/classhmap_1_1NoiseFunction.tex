\doxysubsection{Detailed Description}
A class for generating noise functions. 

The {\ttfamily \doxylink{classhmap_1_1NoiseFunction}{Noise\+Function}} class provides an interface for generating noise-\/based functions. It inherits from the {\ttfamily \doxylink{classhmap_1_1Function}{Function}} class and adds parameters for frequency scaling ({\ttfamily kw}) and a random seed ({\ttfamily seed}) to customize the noise generation.
