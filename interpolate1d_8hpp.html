Detailed Description

Defines a 1D interpolation class using the GSL (GNU Scientific Library).

Author
* Otto Link (otto..nosp@m.link.nosp@m..bv@g.nosp@m.mail.nosp@m..com)

This file contains the Interpolator1D class and InterpolationMethod1D enum, which provide a simple interface for performing 1D interpolation with various methods available in the GSL. The interpolation methods include Akima, cubic, linear, polynomial, and Steffen interpolation.
